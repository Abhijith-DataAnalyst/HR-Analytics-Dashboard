# HR Analytics Dashboard - Tableau

## Project Overview
This HR Analytics Dashboard built with Tableau provides comprehensive insights into key human resource metrics to help HR professionals and business leaders monitor and make data-driven decisions on employee retention, attrition, recruitment, and overall workforce management.

## Features
- Interactive visualizations on employee count, attrition rate, job satisfaction, and demographics
- Trendlines to track progress towards HR goals
- Department-wise, gender-wise, and age group-wise analysis
- Metrics on employee performance, training, and career development
- Filters for drilling down into specific segments such as department or location

## Technologies Used
- Tableau Desktop for dashboard creation and visualization
- Data preprocessing done using Excel or similar tools
- Statistical analysis for insights and trend identification

## Data
- Dataset includes employee demographics, job roles, attrition status, satisfaction levels, and performance metrics
- Data cleaning and validation performed prior to dashboard development

## How to Use
1. Open the Tableau workbook file (.twb/.twbx) in Tableau Desktop.
2. Review and interact with visualizations using available filters and drill-down options.
3. Use the dashboard insights to make informed decisions related to HR policies and employee management.

## Project Structure
- `HR_Analytics_Dashboard.twbx` – Tableau workbook file with the dashboard
- `data/` – Folder containing the dataset files (CSV, Excel)
- `README.md` – This documentation file
